# 🚀 Arquitectura End-to-End: Logística Ketoz

Este proyecto implementa una solución integral para la clasificación y costeo de envíos de repuestos en Bogotá, optimizando la toma de decisiones para el CEO y el área administrativa.

## 📂 Estructura del Proyecto
* **Backend:** 4 módulos Python (POO) con validaciones para ingreso de peso y destino.
* **Base de Datos:** SQLite (`logistica.db`) con persistencia de datos de envíos.
* **Analítica:** Dashboard en Power BI con arquitectura relacional para visualización de costos y volúmenes por destino.
